# Marks and Records

![CI](https://github.com/godcrampy/marks-and-records/workflows/CI/badge.svg)

`mnr` is a human approach to productivity. Every once in a while you add a `mark` which is a update of what you are upto. You can keep track of your productivity and mood in this realease.

## Built With

- [React](https://reactjs.org/)
- [Typescript](https://www.typescriptlang.org/)
- [Sass](https://sass-lang.com/)
- [Bulma](http://bulma.io/)
- [Surge](https://surge.sh/)
- [Jest](https://jestjs.io/)
